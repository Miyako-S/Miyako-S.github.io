 DEEN LIVE JOY-Break 4 〜Soul inspiration〜
1999/10/3 （日）　→ 鹿児島市民文化ホール

* Report : saoriさん
 

DEENのMCを思い出せる範囲でまとめてみました。台詞になっていたり、説明になっていたり、私の心の叫びがはいっていたりと、とても読みにくいかもしれませんが、少しでも鹿児島のライブの様子を感じとってもらえると嬉しいです！


10月3日(日)　快晴！！

この日はMLで知り合った、kyoさんとまさきさんにお会いすることができました。ツアートラックの前でしっかり記念撮影！ 席は前から12列目の左端から3番目。でも、よ〜〜く見えましたよ！
　
池森さんと山根さん、じゃれながらトークが始まりました。

池森さん : いやぁ〜〜、鹿児島来ましたねぇ〜〜。どうですか？山根さん？
山根さん : そうですねぇ〜。
池森さん : いやぁ〜、山根さんどうですか？
山根さん : そうですねぇ〜〜。
池森さん : 山根さん、お願いしますよ！ 切れの良いトークを期待してますよ！
山根さん : 「いやぁ〜、だって急にふってくるからぁ〜。

その後も山根さんの 「そうですねぇ〜。」 はかなり続きました...。会場大笑い！！


■池森さん
Break1で初めて鹿屋(かのや)市を訪れた時とても緊張していたけれど、幕が上がったと同時に あたたかい拍手で迎えてもらって、緊張が全部飛んで行ったそうです。まずはそのことに対してお礼を...。それ以来、鹿児島が大好きになって毎年鹿児島には訪れているとのこと･･。(会場大拍手)

タモリさんがする ♪チャン チャン チャン♪ を...。大成功した池森さん、とっても嬉しそ〜〜う。


■宇津本さん
毎回恒例の第一声 「おじゃったもんせぇ〜〜。」 ※おじゃったもんせ : 鹿児島弁でよくおいでくださいました、と言う意味です。

宇津本さんが俳句を作るきっかけとなったのは、Break1で初めて鹿児島を訪れたときに、JRで鹿児島の西方（にしかた）を通ったときだったそうです。西方のきれいな海と白い雲がとても印象的だったとか。そこで宇津本さんが初めて作った俳句のお披露目。

『うろこ雲 西方の海 つばめの背中』

ここでのつばめは、鹿児島〜福岡間を結ぶ特急つばめの意味を含んでいるとのこと...。(※この話は会報にも載っていますよね...。ご存知の方ばかりだとは思ったのですが･･。書いてしまいました。)

池森さん : 季語はないの？
宇津本さん : う〜〜ん。基本的にそういうのにはこだわっていません。俳句と言うより川柳ですから･･。


■田川さん
前日の10月2日の夕方に鹿児島入りしたことをまずは教えてくれました。(会場はえ〜〜〜っ！) 鹿児島に到着してすぐにフェリーに乗って、一人で桜島に行かれたそうです。でも、もう薄暗くて あまりよく見えなかったとのこと...。10分足らず桜島を眺めていたそうです。

田川さん : 遅かったんで，フェリー乗り場にも誰もいませんでしたよぉ〜。お土産やさんにもだ〜〜れもいなくて...。なんか捕っちゃおうかな？と思ったんですけど...。もちろん捕っていませんよぉ〜。」


■山根さん
桜島に行ったのでその日鹿児島市の繁華街、天文館（てんもんかん）で行われた食事会にはずいぶん遅れて参加したとのこと...。遅れてきたことを池森さんにつっこまれていました。

池森さん : 山根さんは何をしていたんですか？
山根さん : 今朝、ホテルの周りをジョギングしました。
池森さん : 何で？
山根さん : 海が見えて桜島を見ていたらなんとなく走りたくなってですねぇ〜〜〜。
田川さん : そのわりにはホテルではエレベーターを使っていましたよね...。
山根さん : ホテルは8階に泊まっていましたので...。
(へぇ〜〜、8階に泊まってたんや...。私... 納得...)

池森さん : どんなかっこうで走ったんですか？
山根さん : 短パンで...。
（一体どんな色の短パンで走ったんでしょうねぇ〜〜。）


■池森さん
池森さん : 本当に鹿児島の皆さんにはいつも温かく迎えてもらって...。今朝、部屋のカーテンを開けた瞬間目の前に壮大な桜島を見てとても感動しました。鹿児島には毎年来てますが、こんなにはっきりとした桜島を見たのは初めてでした。いつも曇っていたりしていたので...。そう言えば、2、3日前に爆発したんでしょう？
(会場シ〜〜〜〜〜〜〜〜ン)
池森さん : あれ？ だまされたかな？ 結構あるんですよねぇ〜。
(確かに爆発したみたいです。...私... 多いにうなづく。)


1日目はこんな感じでした。その後の感じは... もうお話ししなくても十分わかっていただけると思います。それはそれは大盛り上がり！！！
