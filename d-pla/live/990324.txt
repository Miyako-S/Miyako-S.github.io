 DEEN LIVE JOY-Break 3 〜The 15 DAYs〜
1999/3/24/ （水）　→ 大阪厚生年金会館　2日目

* Report : meo
 

ちょっと違う観点でレポート。大阪ライブ 2日目です。


■ライブグッズ
この日は、前の日時間がなくって 1つも買わなかったライブグッズをすべて購入しようとちょっと早めに会場に入ったんですが、ウワサの 「お箸」！ GETできなかったんですぅ〜！(;_;) 大阪限定発売だったらしいという...（ホントか？？？） グッズ販売スタッフの人たちが立っている後ろの壁には 「スポーツタオル200個限定発売 (SOLD OUT のシール付き...)」 などと書いてありましたが、大阪はお箸だったと聞いたんだけど...？ スポーツタオルだった会場は一体どこなんでしょう？ しかし、1日目にお箸をGETしていた友達...。前の日ちゃんと見てればよかった...。この日は、入り口でお兄さんにもらった広告にはさまってたメールオーダー分以外のグッズなんか目に付かなかったのだ。ううっ。でも買えるグッズは全部買いました。Ｔシャツは2色とも購入、暖かくなったら彼氏とお揃いで着ようかな♪ キャップも 2つになったしね。


■フラッシュ
あんまり書きたくないんですけど、あえて書きます。予定時間より15分早く会場になったんで、グッズがゆっくり買えるや、と思って、とりあえず自分の座席に荷物を置きに行ったんです。「さあ、グッズ♪」 とホールを出ようとした時、フラッシュが...。入り口で検査されたはずなのに... ちょっとショックでした。その直後、警備のお兄さんたちが目を光らせて会場をウロウロし始めたのは言うまでもないことです...。開演前で人もまだまばらだった時だったんですけど、カメラを持ち込んでライブ中に写真を撮ろうとか思ってたんでしょうかね？ そりゃ〜、私だってできるもんならそうしたいですけどマナーは守らなきゃいけないぞっ！ と思ったライブ直前... でした。


■MC...
1日目のMCはなんとか記憶を引っぱり出しましたが、2日目、果たしてどこまで覚えているでしょうか？(＾_＾;

まずはUzさん。
「昨日は予告通り、大阪城に行こうと思ったんですが、道がわかんなかったんで人に訊いたんですよね。したら 『梅田に行け』 っていうんですけど 『梅田駅』 ってたくさんあるんですよ（笑）。いくつあるんですか？ で、結局わかんなかったんで、予定を変更して心斎橋とか、道頓堀あたりに行きました」

巨匠・田川さん。

巨匠 : 大阪と言えば 『たこ焼き』 ということで、僕は昨日はたこ焼きを食べに行きました。で、家でもたこ焼きを作ろうじゃないかと思ったんで、 あの〜、たこ焼きを作るやつ... 『たこ焼き... 製造機』？ あれをですね...。
池森さん : 『製造機』 って...（笑）。
会場 : たこ焼き機ぃーー！ ←おもいっきり関西弁で。
巨匠 : あ、『たこ焼き機ぃ〜』 (笑)ですか。それ買ったんですよ。5回ぐらい作ればもと取れるでしょ。形はね、いろいろあったんですけど... あ、ピカチュウとかありましたよ。
池森さん : ほ〜っ！　ピカチュウ！
巨匠 : でも、あんなのかわいくないですよ、焼くとプク〜ってふくらむんですよ？（笑）

リーダー・山根さん。

リーダー : 今日は朝7時に目が覚めたんで、8時に部屋を出て、僕うどんが食べたかったんですけどね。あ、今のなんかDEENらしくない発言ですね？（笑）」
池森さん : 『DEENらしい』 って... どういうことが 『らしい』 の...(笑)。
リーダー : いや、芸能人らしくないっていう...(笑)。で、そんな朝早くだと、そういううどん屋さんって開いてないじゃないですか？ だから早くから開いてる立ち食いそば屋さんでうどん、食べてきました。

立ち食いそば屋というと、「都そば」とかでしょうか？（スタンダードすぎ？？？ 笑） 私、立ち食いそば屋さんでもしリーダーの後ろ姿を見かけてたとしても気付かなかったんじゃないかって思っちゃいました...。(＾_＾;

リーダー : で、きつねうどん？ 食べましたよ。（←たぶん... ね）

などなど、いろいろ話してました。まだまだ話が続きそうだったんだけど...。

池森さん : あ、山根さん、時間、時間が押してるんで...(笑)。
リーダー : え！？　あ、終わり？？？ まだオチまでいってないのに... もう終わりですか(笑)。
池森さん : はは、オチがあったんですか？ その話...(笑)。
リーダー : いや、なかったんですけどね...。フォローありがとうございます。......それ言われなかったらどうしようかと思ったよ（笑）。

そして池森さん。

池森さん : 髪切っちゃいました、Vocal. 池森秀一です！

その後は、なんだかまじめに語ってたのであんまり覚えてません...。（何じゃそりゃ。笑）

■ソデソデダンサーズ
今回も楽しく見させていただきました、ソデソデダンサーズの皆さん...。「ひとりじゃない」のサビの前、「君の家が見えてき〜た〜 足早になっていく♪」 のところで腕を交互に、体の前で上下に振る踊りを披露していただいて、私たちは真似して踊ってました(笑)。振り返ると、会場のいろんなところでいろんな人が同じことやってました。もうどの曲だったかは覚えてないですけど、かなりおかしな振り付けもあったみたい...。(＾_＾;


■アン・ソデソデダンサーズ（笑）
ソデソデさんのことを書いていて、思い出したこと。これは大阪1日目のことなんですけど、私より前の方（6列目より前）でやや中央寄りの女の子がひとり、歌詞に合わせて踊っているのを目撃してしまいました。

ソデソデさんたちのようなものじゃなくって、例えば 『手ごたえのない愛』 の2番、「街じゅう〜走り〜 虹を探した〜♪」 ってところでは、まず 「街じゅう」 で、両手を左右に広げる動作をして、「走り〜」 で、駆け足の激しい動きを見せ、「虹を」 では、やはり両手で虹の絵を表すような動きをして、「探した〜」 では、そうです、右手を目の上にかざしてキョロキョロするんです。そのまんま、『歌詞に合わせて踊る』 んです。(*_*;

私それを発見してしばらくは、DEENどころじゃなかったです（笑）。池森さんが歌っているのを見たいのに、目がどうしてもアン・ソデソデのお姉さんに向いてしまう...。


■「うぉううぉう♪」
1日目はステージに向かって左側、2日目は右側の座席。「昨日はここで 『うぉううぉうチーム』 やったし、今日は 『ホリデーホリデーチーム』 やな♪」　と勝手に決め込んでいた私...。見事に裏切られ(？)、2日目も 「うぉううぉうチーム」 でした。

まぁそれはどうでもいいんですが（笑）、山根先生（笑）と池森さんに 「いぇいいぇい」 のすばらしいお手本を披露していただいている間、待ち組の私は、近くに見えていた巨匠（田川さん）を見てたんです。ず〜っと見てると、巨匠、なんだか心地よく体をゆすって踊っていらっしゃる...。しかもソデソデさんたちと同じ踊りを...（笑）。(両手を両脇で広げて上下に動かすっちゅ〜やつですね。) なんかそれが異様にかわいく見えたんですね、私には。よく見るといい男...v(←こらこら。(＾_＾;)　いやいや、演奏をやめて巨匠の隣にやって来て元気に踊るUzさんもかなり貴重でしたが、ワタクシ、このたび巨匠ファンになりました。田川さんフリークな方々、これからよろしく。m(_ _)m

■犯人は誰？
『Memories』 の前奏が始まって池森さんが前屈みになった時、演奏がピタッと止まりましたよね？ あれって何だったんでしょう？ 前の日は、前奏に続いてそのまま池森さんが英語の歌詞（？）を歌い始めた（貴重！）んですが......？

池森さん : ...あの〜、これちょっと... （焦っている様子...）
Uzさん : （ズダダン、っと軽くドラムを鳴らす）
巨匠 : （ポロロン、とこちらも軽く流すカンジ） ニヤッと笑う。
会場 : ......？ （そしてどわ〜っと騒がしくなる）
池森さん : あのね、ちょっとね、（とかなんとか言いながらステージの上をウロウロ）
池森さん : う〜、落ち着け！（と自分に言い聞かせている？）

何か間違ったんでしょうか？ よくわかんなかったですけど。そして池森さんは言い訳をするように喋り続けます。

池森さん : いつもはっていうか、今までBreak1からやってきてこんなことなかった、っていうか あっちゃいけないんだけど、あの〜、なんか、ツアーもあと何ヶ所か残すだけになって、何て言うの、あの〜、帰りたくないなっていう...。名残惜しいカンジ？（苦笑） ホント、涙がポロッとこぼれそうだったもん。（と、爽やかなスマイル...♪）
会場 : キャ〜！ 帰らんといて〜！(>_<) ( と黄色い声...)
池森さん : あの〜、みんな、家に帰ってから人に、『ボーカルの人泣きそうになってた』 って言わないように（笑）。

このハプニングがあったせいで、池森さんはラストに 「今日は忘れられない日になりそうだ（笑）」 って言ってました。(＾_＾; 今までやってきて、一度もそういうことなかったのかな？ この時、めちゃくちゃ焦ってたようで、まったく落ち着きのなくなった池森さんがかわいかったv　※あとでわかったことですが、これは機材のトラブルだったらしいです。

-------------------------

私はこういう視点でライブを楽しみました♪ 曲については触れてませんが、伊勢原ライブと同じです。すごくよかったですよ、演出とかも。私も 『VOYAGE』 の最後、照明がはじけたのにはビックリしました。DEENは音がすごく好きです、DEENが奏でる音... ですか？ 心地がいいっていうか、温かみがあるんですよね。これからも 「DEEN」 としての形は変えることなく、いろんな変化をしていってくれたらな、とファンも期待してます。Break4が待ち遠しいっ！(←気が早すぎ？笑)
