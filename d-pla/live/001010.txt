【えりちゃん的ライブレポ 〜鹿児島編〜】

10/10(火)、本日は晴天なりぃ〜！！ DEENが来鹿する時は必ずいいお天気！この日も とってもいいお天気だったのですが、鹿児島の街は、3日前の10/7(土)に 桜島が5年ぶりの大噴火をしたので、灰でかなりグレーに染まってました。

私は、仕事帰り... という事もあって、制服でライブに行っちゃいました。歯科助手なので、ナース服がピンクになった服だと思って頂ければいいです。あと、花柄のエプロンね！ほんとはね〜、“慎吾ママ”の格好をして行きたかったんだけど、ロリータファッションと間違えられそうだったので、諦めました。

「鹿児島ライブは熱いっ！」 と言われていますが、今回悲しい事に、あんまり盛り上がってなかった気がするのは、私だけじゃないはずです。毎年、立ち見まで出るのに、2階席は4人しか入ってなく、1階の後ろも空席がある様な状態でした。

登場の仕方は 前と何も変わってないです。『YOU GATTA KICK』 でメンバー 1人ずつ登場です！私は今回 Break-5 は、伊集院・鹿屋・熊本・鹿児島... の4公演の参加だったのですが、実は私、最初に登場するあのシルエットの池森さんを、鹿屋公演の時まで本物の池森さんだと思ってたんですぅー！鹿屋公演の時に、「池森さん、凄いね〜！トリックだぁ〜！！」 って、隣の席の友達に言ったら、「信じられんっ！騙されてる人がここにいるよ！！」 と、びっくりされたのです。

おっと、話がずれてしまいましたので元に戻しますね。

『Memories』 『すてちまえ！』 と、テンポのいい曲が2曲続く。池森さんのダンス(あれ、ダンスだよね。笑)凄いかわいいっ☆ その後、『このまま君だけを奪い去りたい』。 鹿児島のスタッフが頑張りすぎなのか、池森さんに当たるライト、強すぎぃ〜！ 私の席(12列目)からは、強いライトのせいで、ステージが見えな〜いっ！ 山根さんと田川さんに関しては、どこにいるのかも分からないくらい･･。『秋桜』 の前奏の池森さんのため息の様な声、あの声に私は弱い！ いつもはCDで聞いてる あのため息、生で聞くと、体半分はもう溶けちゃったよ〜。(*＾＾*) 『風の中に』 『Teenage dream』 『夢であるように』 と、バラードが続く。

ここから、アコースティックコーナーに入ります。曲は、『愛があるから』 『恋人よ、夢も嘘もすべて』 『翼を広げて』 『TAKE OFF!』 の順。トーク部分を教えちゃうねっ！

池森 : 鹿児島に来ると 毎回 「帰ってきた感じだ」 って言ってるんですけど、今回は スタッフのみんなまで 「帰ってきた感じがするね」 って言ってたんですよぉ〜！」
会場 : おかえりぃ〜！！(パチパチ・・拍手)」

＊＊＊＊＊＊＊＊＊　中略　＊＊＊＊＊＊＊＊＊＊

池森 : 8/8・8/9 に伊集院と鹿屋に来たんですよね。
田川 : そこに来てない方は、Break-4 の 10/8 ぶりですからね。1年経ってますから、お久しぶりです。
田川 : 昨日は熊本から特急の“つばめ”で鹿児島に入りました。いつもは九州の場合車で移動なんですけど、僕たちだけ昨日は電車で、スタッフは後でバス移動だったんですよ。
池森 : 移動バスを 「ジョイバス」 っていうんだけど、僕達の乗るのが 「ジョイバス1号」 で、スタッフが乗るのが 「ジョイバス2号」。「2号」 はきっと居酒屋状態でしょうけどね〜(笑)。
田川 : そのバスの運転手さんがね、「DEENは よく鹿児島に来るけど、鹿児島が好きなんだねぇ〜」 って言ってたらしいんですよ(笑)。
(この時田川さんは、酔っ払い口調でしゃべりながらバスを運転するジェスチャーをしてる。)

ここでちょっと、会場の〔鹿児島市民文化ホール〕を説明しときますね。この鹿児島市文には、第一ホールと第ニホール、隣り合わせで2つのホールがあります。第一ホールの方が広い会場です。

田川 : ここ、第一ホールですけど、前回は第ニホールでしたよね。
池森 : 前回だっけ？ 僕たちが第ニホールでライブをやってる時に、隣り(第一ホール)で美川憲一さんがコンサートしてたの(笑)。
田川 : そうですね。隣りの美川さんのコンサートに来た人が間違ってこっちに入ってたりしてね〜(笑)。
池森 : 最後までDEENのライブを見て、「あれっ？ 美川さんのコンサートじゃない」 って気付いたり･･ するわけないってね(笑)。
田川 : DEENは Break-1 からこの文化ホールでライブをしてますが、第一ホールと第ニホール、順番にジグザグにしてますよね。

田川 : ついこの前ですか？桜島が大噴火したの、いつでしたっけ？
会場 : 土曜日ぃ〜！！
池森 : そう、土曜日ですね。
田川 : 桜島が噴火して、鹿児島市内の方に 桜島の灰がかなりきたらしいですね。僕たちはそういうの経験した事ないからあまりよくわからないんですが、こっちのイベンターさんが言うには、「あ〜、来る、来る！」 っていうのが分かるんですってね！ その桜島の爆発で車のガラスとか割れちゃったりして、この前は大変だったみたいですね。
会場 : うん、うん(と うなずく。)
田川 : まぁ... ね、それを含めて鹿児島！ということですから、噴火したからって鹿児島に来ない訳じゃないですしね。桜島の灰も含めて鹿児島は大好き！ ...変なまとめ方ですけど...(笑)。

池森 : DEENリーダー、キーボード・山根公路！
(なぜだか分からないけど、会場は山根さんを見ながら大爆笑！)
池森 : なんでみんな笑うの？ みんな、山根さんを芸人さんだと思ってないですか？(笑)
山根 : ...っていうか、DEENは2人じゃないんだからさぁ〜...。僕が言おうとしたことを全部2人でしゃべっちゃって！ 今僕は、「あ〜これも言われた、またこれも消えた」 って、何をしゃべろうか一生懸命考えてたんですよ！ だって僕、今まで一言もしゃべってないでしょ？
池森 : うん、しゃべってないね（笑）。でも、山根さんは今日特別な事があったじゃないですか！(笑)
山根 : そう... 今日ね、ホテルのロビーに集合が2時だったんですよ。でもね、僕の部屋の時計が 1時間遅れててね〜、それで、コンビニにお昼ご飯でも買いに行こうと部屋を出たのがPM12:30だったんですね。でも、それはすでにPM1:30だったわけですよ！
池森 : 僕はそのPM1:30に 山根さんがコンビニ行くのを見てるんですよ！(笑) その時僕は、「山根さんまさか時間間違えてないよなぁ〜」 と ちょっと思ったりもしたんですが、大丈夫だろう... って自分の部屋に戻ったんですね。で、メンバーやスタッフは、だいたいPM1:55頃になったら車に乗り込む訳ですよ。PM2:00になっても山根さんが来なくて、スタッフもマネージャーも 焦っちゃって、「山根さんが！山根さんが！」 とか言いながら走り回ってるわけですよ。山根さんの携帯にかけても出ないし、ホテルの部屋に電話しても出ない、とうとうマネージャーが ホテルの人に言って 山根さんの部屋に一緒に行ってもらって、鍵を開けてもらったんですよ。その時マネージャーは、「もしかしたら部屋の中で変な事に...」 なんて 変な想像をしてたらしく、山根さんの部屋のカギを開ける時、心臓がバクバクしてたらしいんですよ！
山根 : そういう時って、何でも考えつくもんですよね〜(笑)。
池森 : そして、鍵を開けてもらって まずベットの上を見て、「ベッドの上ではない」 と安心したみたいなんだけど、「もしかしてお風呂で...」 なんて考えついて、お風呂を開けたけど お風呂にもいない。「まさか山根さん、ベランダで寝てるんじゃないだろうな〜」 とまで思ったらしいんだけどベランダにもいなくて、常に 僕たちが待っている車の中と連絡を取り合ってたんですけど、車の中では 「山根さんはたぶん、集合時間をPM3:00だと勘違いしてるんだよ！」 という話になってたんですよ。そして、車の外を見ていたら、山根さんがゆ〜っくりコンビニの弁当なんかを下げて歩いてるじゃないですか！！

この話の最中、山根さんは凄く恥ずかしそうにしてた(笑)。池森さんは爆笑しながら話してる(笑)。

池森 : みんなで笑いながら 「公ちゃ〜ん、何してるのぉ〜？集合の時間だよ！」 って言ったら、山根さん 「まだ PM1:00 だよ！」 とか言っちゃってるわけ！ またそこで、みんなで大笑いしちゃって(笑)。
山根 : コンビニで弁当買って 「ゆっくり食べて シャワーでも浴びてから会場入りしようかな〜」 って思ってたのに、反対に汗かきながらバスに乗り込んだよ...。ホテルの人も気をつけて欲しいですよね。掃除するときにでも 確認してくれればいいのにぃ〜！
池森 : ホントだね〜(笑)。

山根さんの話は終わったのに、池森さんはすっかり笑いのツボにはまってしまった！ 肩を振るわして、おなかをかかえて、「はまっちゃったよ〜」 と、ずっと笑ってる。やっと落ち着いて次へ... と思ったらまた、プッ！と ふきだしてなかなか笑いが止まらなかった(笑)。会場は、山根さんの話＋池森さんの笑いで大爆笑！！（笑）

アコースティックコーナーが終わり、『Long Distance』、新曲 『悲しみの向こう側』 と続く。『悲しみの...』 では私涙うるうる... とまではいかなかったけど、かなり、ジ〜ンときたよ。池森さんの歌詞の世界って、何て素敵なのかしらぁ〜！
